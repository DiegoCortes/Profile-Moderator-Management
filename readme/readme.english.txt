[size=4][color=#bf307a]Introduction[/color][/size]
[size=1em]Profile Moderator Management provides a very simple way to grant moderator privileges to your users in specific boards.[/size]

[size=3][color=#1f4882]Features[/color][/size]
[list]
[li][size=0.9em]Remove/Add the user as moderator from their profile[/size][/li]
[li][size=0.9em]Select multiple boards at once[/size][/li]
[li][size=0.9em]Select entire categories[/size][/li]
[/list]
